Для небольшой цитаты используется тег <q>, для уже серьезной тег<blockquote>. Чтобы указать источник или автора используется <cite>

По словам Чарльза Буковски — Интеллектуал о простой вещи говорит сложно — художник сложную вещь описывает простыми словами.(с использованием тех тегов)

Теги <em> и <i> используются для акцентирования внимания(выделяются курсивом). Первый меняет смысл предложения, а второй не является таким важным, но все равно выделяется, это могут быть мысли героя, либо что-либо др.

Теги <strong> и <b> делают слова жирными. Первый для привлечения внимания, чтобы люди читали его первым, он важный. А второй тоже привлекает внимание, но не такой важный

Далее идут теги <div> и <span>. Как я понял, они нужны лишь для выдачи какому-то тексту или фразе класса. Первый для большого текста, второй для фразы.

Для обозначения времени используется <time>. Он в основном нужен лишь самому компьютеру.

                        09:54 утра
                        18 ноября 2015
                        в прошлую субботу
                        вчера

                        <time datetime="2016-11-18T09:54">09:54 утра</time>
                        <time datetime="2015-11-18">18 ноября 2015</time>
                        <time datetime="2018-09-23">в прошлую субботу</time>
                        <time datetime="2017-04-20">вчера</time>

Для каких-то химиков математиков есть теги <sub> и <sup>. Это чтобы степени делать. Типа если b в конце, то циферка внизу, если p, то вверху. Все элементарно.

И легенькое на десерт, теги <ins> и <del>. Они нужны, чтобы показывать наглядное дополнение списка, либо просто чтобы было. Первый подчеркивает, второй перечеркивает. Обычно дополняются атрибутом datetime

Пример:

  • Сделать дз
  • Поменять толчок коту
  • Поспать
  • Стать долларовым миллиардером
                        <ul>
                            <li>Сделать дз</li>
                            <li><del datetime="2020-10-13T01:25-07:00">Поменять толчок коту</del></li>
                            <li><ins>Поспать</ins></li>
                            <li><ins>Стать долларовым миллиардером</ins></li>
                        </ul>

Тег <img> отвечает за добавление картиночек. Сам по себе тег бесполезен без своего атрибута src, который указывает путь к картинке.

<img src="kotik.png">

Можно добавлять несколько типов картиночек.

  • SVG

    Подходит, если:

    • необходимо масштабировать изображение без потерь
    • изменять цвет элементов изображения
    • нужно анимировать части изображения
  • JPEG

    Подходит для:

    • полноцветных изображений, фотографий
    • изображений с плавным переходом яркости и контраста
    • рисунков с большим количеством разноцветных деталей
  • PNG

    Подходит для:

    • изображений с прозрачностью и полупрозрачностью
    • полноцветных изображений, когда необходима повышенная точность
    • изображений с резкими переходами цветов
  • GIF

    Подходит если:

    • нужна простейшая анимация

Тег <figure> подходит для любого иллюстративного или демонстрационного материала, которым можно дополнить содержание документа: схемы, графики, примеры кода, таблицы и так далее.
При удалении такого материала основное содержание не должно пострадать, иначе это не дополнительный материал и тег <figure> не подходит для его разметки.
Обычно в дополнении к этому тегу, добавляют <figcaption>, он нужен чтобы просто добавить описание.

Если например, человек перешел по ссылке на фоточки, и он находится на первой, то кнопку Назад можно сделать пустой ссылкой(необязательно назад, хоть где крч).
<a>Назад</a>
